# TepsLab 프로젝트 비전

## 한 줄 정의
TEPS 시험 준비를 위한 프리미엄 온라인 학습 플랫폼

## 대상 사용자
- TEPS 시험을 준비하는 수험생
- 327점, 387점, 450점, 550점 등 점수대별 목표를 가진 학습자
- 체계적인 학습 커리큘럼이 필요한 사용자

## 해결하는 문제
1. 기존 TEPS 학습 자료가 점수대별로 맞춤화되지 않음
2. 학습 진도 추적 및 취약점 분석이 어려움
3. 모바일에서 학습하기 불편함

## 핵심 가치 (3개)
1. **맞춤 학습** - 점수대별 커리큘럼으로 효율적인 학습
2. **데이터 기반** - 학습 진도 추적 및 취약점 분석
3. **접근성** - 모바일/데스크톱 어디서나 학습 가능 (PWA)

## 핵심 기능 (MVP)
- [x] 사용자 인증 (이메일, 소셜 로그인)
- [ ] 점수대별 코스 제공
- [ ] 학습 진도 추적
- [ ] 모의고사 기능
- [ ] 결제 시스템

## 🚫 목표 NOT (명시적으로 하지 않을 것)
- 실시간 화상 강의 (녹화 강의만)
- 1:1 튜터링 매칭
- 커뮤니티/포럼 기능
- 다른 영어 시험 (토익, 토플 등) 지원
- 네이티브 모바일 앱 (PWA로 대체)

## 성공 지표
- 페이지 로딩 3초 이내
- 모바일 사용자 경험 점수 90+ (Lighthouse)
- 학습 완료율 추적 가능
- 결제 전환율 측정 가능

## 기술적 제약
- Vercel 무료 티어 + Supabase 무료 티어로 시작
- 솔로 개발자가 유지보수 가능한 수준의 복잡도
- TypeScript strict mode로 타입 안전성 확보
